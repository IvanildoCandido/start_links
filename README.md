# Start Academias - Landing Page

Projeto pronto para deploy estatico na Vercel.

## Publicar na Vercel (mais rapido)

1. Acesse [https://vercel.com/new](https://vercel.com/new)
2. Importe esta pasta/projeto
3. Framework Preset: `Other`
4. Build Command: deixar vazio
5. Output Directory: deixar vazio
6. Clique em **Deploy**

## Estrutura

- `index.html` - pagina principal
- `styles.css` - estilos da pagina
- `assets/` - imagens (logo e fundo)
- `vercel.json` - configuracao de deploy/roteamento/cache
